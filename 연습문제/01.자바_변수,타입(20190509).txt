자바 변수, 데이터 타입

1. 원의 면적을 구하는 프로그램을 작성하기

순서1) 프로젝트명과 클래스명은 직관적으로 생성
순서2) 원주율(PI)을 저장하는 실수형 상수 변수를 생성하고 값을 설정 (PI = 3.14)
순서3) 원의 반지름을 저장하는 실수형 변수를 생성하고 적당한 값을 설정
순서4) 원의 면적을 출력하기 (반지름 * 반지름 * PI = 원의 면적)

2. 성적표를 구하는 프로그램을 작성하기

순서1) 프로젝트명과 클래스명은 직관적으로 생성
순서2) 해당 변수를 생성하고 초기값은 0으로 설정하세요. (변수명은 직관적으로 작성 바람)
국어, 영어, 수학, 과학 정수형 변수 생성.

순서3) 국어 점수를 90으로 설정하고 점수를 출력.
순서4) 영어 점수를 80으로 설정하고 점수를 출력.
순서5) 수학 점수를 70으로 설정하고 점수를 출력.
순서6) 과학 점수를 60으로 설정하고 점수를 출력.
순서7) 모든 과목의 평균값과 총점을 저장할 변수를 생성. (변수명은 직관적으로 작성 바람)
순서8) 평균값과 총점을 출력하세요.

3. 알파벳과 한글의 자음, 모음을 구하는 프로그램 작성하세요.

순서1) 프로젝트명과 클래스명은 직관적으로 생성하세요.
순서2) 알파벳 자음, 모음과 한글 자음, 모음에 대한 정보는 인터넷을 검색하여 정보를 획득 바람.
순서3) 알파벳 자음, 모음과 한글 자음, 모음 변수들을 생성하고 초기값을 0으로 설정하세요.
순서4) 각 변수에 대해서 자음, 모음 갯수를 입력하세요.
순서5) 알파벳 자음, 모음과 한글 자음, 모음에 대한 갯수를 출력하세요.

4. 온도를 구하는 프로그램 작성.

순서1) 프로젝트명과 클래스명은 직관적으로 생성하세요.
순서2) 화씨 온도를 저장하는 정수형 변수 생성.
순서3) 화씨 온도에 40도의 값을 설정.
순서4) 섭씨 온도를 저장하는 실수형 변수 생성.
순서5) 화씨 온도를 섭씨 온도로 변환하세요. (화씨온도 - 32) / 1.8 = 섭씨온도
순서6) 섭씨 온도를 출력하세요.

5. scanner 입력에 대한 연습이 충분하면 아래의 프로그램을 만드세요.

순서1) 프로젝트명과 클래스명은 직관적으로 생성하세요.
순서2) 문제1 에서 원주율을 저장하는 변수를 사용자가 입력하여 면적을 출력.
순서3) 문제2 에서 국어, 영어, 수학, 과학 점수를 사용자가 입력을 하고 평균값과 총점을 출력.
